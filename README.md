# Daily Productivity Dashboard

A simple web app to:
- Add daily tasks
- Track current date & time
- Get weather updates for any city

### Tech Stack:
- HTML
- CSS
- JavaScript
- OpenWeather API

### How to run:
1. Clone this repo
2. Open `index.html` in your browser
3. Get a free API key from https://openweathermap.org/ and replace it in `script.js`
